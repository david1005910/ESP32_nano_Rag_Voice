## ⚙️ 슬라이서 설정

### 공통 설정 (Cura / PrusaSlicer)

|------|-----|
| 레이어 높이 | 0.2mm |
| 벽 두께 | 1.2mm (3줄) |
| 상/하 두께 | 1.2mm (6층) |
| 인필 | 20% |
| 인필 패턴 | Gyroid 또는 Grid |

## 🔩 조립 순서

### 시계형 조립
```
1. 04_speaker_holder에 스피커 삽입
2. 01_case_body 안에 스피커 홀더 배치
3. 03_internal_tray에 ESP32, 마이크, 앰프 고정
4. 트레이를 케이스에 삽입
5. LCD 연결 및 장착
6. 02_case_back으로 닫기
7. (선택) 05_watch_lugs 부착
```

### 스탠드형 조립
```
06 또는 07 스탠드 사용:
1. 스탠드 내부에 부품 배치
2. 배선 연결
3. LCD 장착
4. USB 케이블 연결
```

## ❓ 문제 해결

### 부품이 맞지 않아요
- 슬라이서에서 "수평 확장" -0.1mm 설정
- 사포로 접촉면 다듬기
- 구멍은 리머나 드릴로 확장

### 서포트가 안 떨어져요
- 서포트 Z 거리: 0.25mm
- 서포트 밀도: 10-15%
- 핀셋, 니퍼 사용

### 출력물이 휘어요
- 베드 레벨링 재확인
- 브림 또는 래프트 사용
- 베드 온도 확인
