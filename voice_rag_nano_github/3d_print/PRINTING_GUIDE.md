## 🔧 OpenSCAD 사용법

### 1. OpenSCAD 설치
```
Windows/Mac: https://openscad.org/downloads.html
Linux: sudo apt install openscad
```

### 2. STL 파일 생성

1. `.scad` 파일을 OpenSCAD에서 열기
2. 파일 하단에서 인쇄할 부품 선택:

```scad
// 아래 줄의 주석(//)을 제거하고 하나만 활성화

// assembly_preview();  ← 미리보기 (비활성화)
case_body();            ← 이걸 인쇄하려면 주석 제거
// case_back();
// internal_tray();
```

3. `F5` → 미리보기
4. `F6` → 렌더링 (시간 소요)
5. `파일` → `Export` → `STL`

| 출력 속도 | 50mm/s | |

### 부품별 설정

#### case_body (케이스 본체)
```

서포트 밀도: 15%
서포트 Z거리: 0.2mm

```

```
브림: 3mm
```

```

```

#### desk_stand (탁상 스탠드)

## 🎨 추천 필라멘트

### PLA (초보자 추천)
```
장점: 인쇄 쉬움, 냄새 적음, 저렴
단점: 열에 약함 (60°C 이상 변형)
색상 추천: 검정, 회색, 네이비
온도: 노즐 200-210°C, 베드 60°C
```

### PETG (실사용 추천)

```
단점: 스트링 발생 가능
온도: 노즐 230-250°C, 베드 70-80°C
```

## 🔩 조립 순서

### 1단계: 스피커 설치
```
1. speaker_holder를 case_body 안에 배치
2. 20mm 스피커를 홀더에 끼움
3. 케이블을 측면으로 빼둠
```

### 2단계: 내부 트레이 설치
```
1. internal_tray를 케이스에 삽입
2. ESP32-S3 Nano를 마운트에 고정 (M2 나사)
3. INMP441 마이크 설치
4. MAX98357A 앰프 설치
```

### 3단계: 배선
```
1. 모든 모듈 배선 연결
2. USB-C 포트 위치 확인
3. 케이블 정리
```

### 4단계: LCD 설치
```
1. LCD FPC 케이블 연결
2. LCD를 상단 구멍에 삽입
3. 베젤이 제자리에 맞는지 확인
```

### 5단계: 마무리
```
1. case_back을 끼워맞춤
2. 필요시 M2 나사로 고정
3. 동작 테스트
```

## ❓ 문제 해결

### 서포트가 잘 안 떨어져요
```
해결:
- 서포트 Z거리 증가 (0.25mm)
- 서포트 밀도 감소 (10-12%)
- 서포트 인터페이스 사용
```

### LCD가 안 맞아요
```
해결:
- lcd_diameter 값 조정 (+0.5mm)
- 인쇄 후 사포로 다듬기
- 슬라이서에서 수평 확장 -0.1mm
```

### ESP32가 안 들어가요
```
해결:
- nano_width, nano_length 값 확인
- 홀더 크기 +0.5mm 조정
- 인쇄 후 리머로 구멍 확장
```

### 케이스가 제대로 안 닫혀요
```
해결:
- 끼워맞춤 공차 조정 (0.5mm → 0.8mm)
- 접촉면 사포질
- 코너 다듬기
```
